# BubbleTasks
BubbleTasks is a cross-platform mobile todo app with time-based alarms built using React Native and Expo. It combines a task management system with time-based reminders using notifications.

## Features
- Create and manage tasks
- Reorder tasks using drap-and-drop
- Schedule time-based reminders via local notifications
- Cross-platform support (IOS and Andriod)

## Tech
- React Native
- TypeScript
- Expo
- Expo Notifications

## Notes
Alarm and the notification behavior depends on the platform's limitations.
Full alarm reliability is best tested using a developemtn build rather than Expo Go
